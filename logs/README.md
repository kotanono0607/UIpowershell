# logsディレクトリ

このディレクトリには、UIpowershellの実行ログが保存されます。

## ログファイルの種類

### 1. サーバー実行ログ
- **ファイル名形式**: `api-server-v2_YYYYMMDD_HHmmss.log`
- **内容**: Polarisサーバーの実行ログ（PowerShell Transcript）
- **生成タイミング**: サーバー起動時
- **自動削除**: 次回起動時に全て削除

### 2. ブラウザコンソールログ
- **ファイル名形式**: `browser-console_YYYYMMDD.log`
- **内容**: ブラウザのconsole.log, console.error等の出力
- **生成タイミング**: ブラウザでUIを操作中（5秒ごとにバッチ送信）
- **自動削除**: 次回起動時に全て削除

## ログの仕組み

### サーバーログ
```
起動時に自動生成 → Start-Transcript で記録 → 終了時に Stop-Transcript
```

### ブラウザログ
```
ブラウザ (console.*) → 5秒バッファ → POST /api/browser-logs → ファイル追記
```

## ログのライフサイクル

1. **生成**: サーバー起動時・ブラウザ操作時
2. **保持**: 現在の起動セッション中のみ
3. **削除**: 次回サーバー起動時に自動削除

これにより、logsディレクトリには常に**直近の実行ログのみ**が保持されます。

## 注意事項

- ログファイルはGit管理対象です（共有されます）
- 機密情報を含む可能性があるため、パスワードやトークンは記録しないでください
- ログファイルが大きくなりすぎる場合は、.gitignoreに追加することを検討してください
